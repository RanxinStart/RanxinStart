# CMD命令指令符


## 目录
+ [cd](#进入指定目录)
+ [md](#创建一个文件夹)
+ [rd](#删除一个文件夹)
+ [dir](#列出当前目录下的文件)

## 使用
+ [进入指定目录](#进入指定目录)
+ [创建文件夹](#创建一个文件夹)
+ [删除文件夹](#删除一个文件夹)
+ [列出当前目录文件](#列出当前目录下的文件)

### 进入指定目录
```CMD
cd 目录名  
```

### 创建一个文件夹
> 在当前路径创建文件夹
```CMD
md 文件夹名
```

### 删除一个文件夹
> 在当前路径删除文件夹
```CMD
rd 文件夹名 
```

### 列出当前目录下的文件
> 列出当亲目录下的全部文件
```CMD
dir
```
